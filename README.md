# vue_nodejs_crud
 CRUD User and Post Using Vue & Nodejs


## Frontend Setup

```sh
cd frontend
npm install
```
## Backend Setup

```sh
cd backend
npm init -y
```
### Run mysql database

```sh
vue_nodejs_crud_db.sql

```
### Server run

```sh
node server.js
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
